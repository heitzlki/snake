## Setup

Clone project

```
git clone https://github.com/heitzlki/snake
```

Change directory

```
cd snake/
```

### CLI-Version

Install dependencies

```
yarn
```

Start script

```
yarn run start
```

### Web-Version

Change directory

```
cd web/
```

Install dependencies

```
yarn
```

Start vite

```
yarn run start
```
